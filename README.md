## About

A simple login & register system built with Nodejs, Express, Passportjs, Mongodb, Passport
It can be used as boilerplate (stater project)

## Supported Auth

- Local
- Facebook
- Discord
- Dropbox

## install

```sh
yarn install
```

## Usage

- Copy `./config.example.json` to `./config.json`
- Set your own keys

## Developement

```sh
yarn dev
```
